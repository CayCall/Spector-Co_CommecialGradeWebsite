UI/UX Design Decisions

The UI/UX design of Spector & Co focuses on creating a clear, professional, and engaging experience. The minimalist design uses a sophisticated navy blue color scheme to build trust and align with the brand identity. Typography features Helvetica for readability, with clear distinctions between headings and body text. Interactive elements like hover effects on buttons and links provide feedback, encouraging user engagement. Finally, white space is strategically used to reduce cognitive load and improve the overall user experience.

Key Design Decisions:

  • Minimalist Design: Simplified layout with a professional navy blue color scheme to enhance brand identity.
  • Typography: Helvetica font for readability, with clear distinctions between headings and body text.
  • Interactive Elements: Hover effects on buttons and links for better engagement.
  • White Space: Thoughtful use to reduce cognitive load and improve user experience.

Folder Structure

In order to maintain a clean and organized project, the folder structure is designed with scalability in mind. Below is an overview of the folder structure used in the Spector & Co website project:

The folder structure is designed to separate different aspects of the project:

  • Images: Contains all images used in the website, including wireframes, logos, and other assets.
  • Styles: Contains all CSS files for styling, including the main styles.css and any additional style sheets for specific pages.
  • Scripts: Houses all JavaScript files used for interactivity, including custom scripts for menu handling, content fetching, etc.
  • Pages: Contains all HTML files for the different pages of the website, keeping everything modular and easy to maintain.
