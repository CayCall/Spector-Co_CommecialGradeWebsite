Specter & Co Style Guide

Inspired by the character Harvey Specter from Suits, this guide outlines the styling choices for a high-end legal consultancy website.

Color Choices

  • Navy Blue (#002B5C): Used for headers, navigation bars, and information cards to symbolize trust and professionalism.
  • Platinum (#E5E5E5): Applied to borders, text elements, and information cards to add balance and readability.
  • Baby Powder (#FFFDF8): A soft, inviting background for sections, input fields, and form areas.
  • Teal (#008080): Used for important buttons and call-to-action buttons like(Below) to signify clarity and decisiveness.
  • Robin Egg Blue (#39CECE): Accent color for hover effects on buttons and interactive elements, adding a friendly contrast.

The selected colors maintain a sophisticated and approachable brand identity. Navy Blue establishes authority, Platinum provides a modern contrast, Baby Powder creates a gentle background, Teal draws attention to key actions, and Robin Egg Blue offers a refreshing touch for interactive feedback.

Button Style

The primary button style uses Teal (#008080) to draw attention to key actions, with a Robin Egg Blue (#39CECE) hover effect for visual feedback. This button styling reinforces both professionalism and approachability.

Typography

Font Choice: Helvetica, a sans-serif font that exudes professionalism and confidence, widely used by brands like BMW and American Airlines.

Consistency: Helvetica will be used across all headings (H1-H6) and body text to maintain readability and a clean appearance.

Line and Letter Spacing

  • Body Text: 22px to 26px
  • Headings: 28px to 34px
  • Subheadings: 23px to 27px
  • Navigation and UI elements: 16px to 20px

Logo and Branding Elements

The logo will feature the name "Specter & Co" using Helvetica font, reflecting professionalism without the need for additional iconography. This fictional logo is created to align with the site's color palette and typography.

Responsive Design

Fluid Typography

  • Mobile: 14px base font size
  • Tablet: 16px base font size
  • Desktop: 18px base font size

Flexible Images and Media

Images will be styled with CSS properties like max-width: 100%; to ensure responsiveness across devices.

Navigation and Interactive Elements

A collapsible hamburger menu will be implemented for smoother navigation on smaller devices.

Media Queries

Media queries will adjust the layout and elements based on device requirements, ensuring a responsive experience across different screen sizes.
